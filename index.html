<strong>Bold, alerting text</strong>
<em>Italic, emphasised text</em>.

CSS

  • Cascading Style Sheets
  • Adding the style to the skeleton.

Think of CSS as the visual arrangement of elements. HTML is largely for computers to read, CSS is for humans to see.

There are three ways of defining CSS.

IDs

IDs are added to a HTML tag via an attribute, and can only be used once per web page. Avoid using these for styling because of this restriction.

Classes

Classes are the most commonly used CSS designator. They can be reused multiple times per page. You can also have more than one class on an element.

Inline

  • Inline styles are used to quickly add style to an HTML markup element.
  • These can be used in the NationBuilder web content editor!
