CSS

  • Cascading Style Sheets
  • Adding the style to the skeleton.


HTML is largely for computers and search, CSS is for humans.

There are three ways of defining CSS.

IDs

IDs are used to style HTML tags when that tag is unique on the page.

Classes

Classes are used to style HTML tags when that tag or block occurs multiple times on the page.

Inline

  • Inline styles are used to quickly add style to an HTML markup tag.
  • These can be used in the NationBuilder web content editor!

Wait! Bold is both HTML and CSS!

<strong>Bold text!</strong>
<div style="font-weight:bold">...Bold text!</div>
